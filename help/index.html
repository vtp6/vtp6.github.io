What is VTP6?

Vocabulary Testing Program 6 (VTP6) is an online successor to the app-based VTP5 program . It provides languages students with a better way to learn vocabulary.

Because it is online, VTP6 can be accessed from mobile phones, while VTP5 cannot. Also, VTP6 has many more features than VTP5, including Match and Quick Fire modes.

VTP6 is also much better than competitors such as Quizlet, because it forces you to do active revision, while flashcards are passive.

Furthermore, VTP6 will always be completely free for everyone, and there are no ads on the site either (Quizlet runs ads, and some features are locked behind a paywall).

How do I use VTP6?

First, navigate to the Folders homepage . Then, select your language from the options there (Latin, Spanish, French, German), and then choose your folder (e.g. GCSE).

From there, you can select which units/chapters of the folder you want to study using the checkboxes. Then, choose your game mode (see below) and press start, then watch your grades shoot upwards!

Which game modes are on VTP6?

Classic Mode

Classic Mode is the most basic game mode on VTP6. A random term from your chosen sets will come up, and you have to type the definition and press ENTER.

You can keep playing until the selected terms run out, or you can finish the game early using the finish button. At the end, you can retry your mistakes.

VTP6 is supported by a superior typo detection algorithm which can prompt you to retry a question if it thinks you made a typo. Furthermore, the algorithm ignores case and punctuation to make it easier for mobile users (where the first letter is usually auto-capitalised).

Classic Infinite Mode

Classic Infinite Mode is an extension of Classic Mode where the terms keep reshuffling forever. To exit, you can press the finish button and you can see your mistakes in a table, just like with Classic Mode.

Match Mode

The aim of Match Mode is to pair up the six terms on the left with the six definitions on the right as quickly as possible. Click a term on the left and then its definition on the right to pair them up. The timer at the top shows you how long you have spent.

Each wrong match adds 1 second to your time. At the end, you can share your score with friends. The site will keep track of your overall best score on Match Mode using local storage in your browser.

Quick Fire Mode

In Quick Fire Mode, you race against the clock to answer as many questions as possible in the time that you are given. Each correct answer gives you some extra time on the clock.

Just like Match Mode, the site will keep track of your overall highest score on Quick Fire Mode using local storage in your browser.

Custom Set File Formats

There are three options for the types of files you can upload as a custom set:

VTP5 Format

VTP6 is backwards-compatible with VTP5 files, which are in the following format: 

Term 1
Definition 1
Term 2
Definition 2
Term 3
Definition 3

VTP6 Format

VTP6 also has its own format for text files, using tab characters to separate the terms and definitions: 

Term 1	Definition 1
Term 2	Definition 2
Term 3	Definition 3

CSV Format

You can also use Comma Separated Value (CSV) files. Note that if a term or definition contains a comma, you should put quotes around it. 

Term 1,Definition 1
Term 2,Definition 2
Term 3,Definition 3

If you have any feedback at all (e.g. a mistake in one of the vocab folders, new game mode ideas, etc.), just email vtp6_feedback@outlook.com.
